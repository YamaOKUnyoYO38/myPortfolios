# fixlist 運用メモ

## 修正箇所を追加するとき（開発者・AI向け）

ユーザーが「修正箇所」を入力した場合は、次を実行してください。

1. **fixlist.txt** の末尾に、次の形式で追記する。
   - 1行目: `- YYYY-MM-DD HH:MM: 修正内容の箇条書き`
   - 入力日時は追記時点の日時とする。

2. 追記後、リポジトリに反映する。
   - `git add fixlist/fixlist.txt`
   - `git commit -m "docs(fixlist): 修正箇所を追加"`
   - `git push origin main`

## 例

ユーザー入力例:
「〇〇画面のラベルを変更したい」

fixlist.txt への追記例:
```
- 2025-02-04 15:30: 〇〇画面のラベルを変更する
```
