# High-Dividend Hunter 機能追加の指示プロンプト

以下の内容を開発者またはAIに指示する際に、そのままコピーして使用してください。

---

## コピー用プロンプト（ここから）

```
# 依頼内容
アプリ「High-Dividend Hunter」に以下の機能を追加してください。

# ルール
- **1機能につき1ブランチ**を作成し、そのブランチで実装する。
- 実装中にエラーや不具合が発生したら**直ちに修正**する。
- エラー・不具合がなく、**アプリが正常に動作する状態が確認できたら**その機能のブランチをマージする。
- マージ後、次の機能はマージ済みの main（またはマージ先）を元に新しいブランチを切り、同様に進める。

# 追加する機能（実装順は任意。1機能＝1ブランチ）

## 機能1: サイト名でランキング取得
- ランキング取得時に、**URL のみではなく「サイト名」のみを指定して取得**できるようにする。
- 例: 登録済みサイト名（「Yahoo!ファイナンス 配当利回り」など）を選択または入力すると、対応するURLで取得する。

## 機能2: 条件指定でのランキング取得
- ランキング取得時に、以下の**条件を指定**して絞り込みできるようにする。
  - 配当率（配当利回り）の範囲
  - 決算月
  - 財務諸表の各項目（個別に指定可能）
  - 業界
  - 分野
  - 株主優待の有無
- 条件は組み合わせ可能とする。

## 機能3: ポートフォリオ機能（リスト・ページ）
- **任意のポートフォリオをリストとして作成**する機能を追加する。
- ポートフォリオ用の**専用ページまたはセクション**を新設する。
- リストの作成・編集・削除ができるようにする。

## 機能4: 銘柄クリックでポートフォリオに追加
- 取得したランキングの**各銘柄をクリック**すると、プルダウン（またはメニュー）を表示する。
- その中から「リストに保存」等を選ぶと、**指定したポートフォリオに銘柄を追加**できるようにする。
- 機能3のポートフォリオ機能と連携すること。

## 機能5: 取得件数の指定
- ランキング取得時に**取得する銘柄の件数を指定**できるようにする。
- **許容範囲: 1件～999件**（この範囲外はエラーまたはバリデーションで弾く）。

# 補足
- 既存の「URL指定でランキング取得」「テーブル表示」「CSVダウンロード」は維持する。
- 技術スタック: Python 3.10+, Streamlit, Requests, BeautifulSoup4, Pandas（変更する場合は理由を明記すること）。
- 不具合やエラーが出た場合は、マージせずに修正し、動作確認後にマージすること。
```

## 使い方

1. 上記「ここから」～「ここまで」のブロックをコピーする。
2. 開発者への依頼文、または AI（Cursor / ChatGPT 等）の入力欄に貼り付けて送信する。
3. 機能ごとにブランチが切られ、動作確認後にマージされる流れで進めてもらう。

## ブランチ名の例（参考）

- `feature/site-name-search`
- `feature/ranking-filters`
- `feature/portfolio-list`
- `feature/add-to-portfolio-from-ranking`
- `feature/result-limit`

必要に応じて、上記プロンプトに「ブランチ名は `feature/〇〇` 形式で作成すること」などを追記して使ってください。
